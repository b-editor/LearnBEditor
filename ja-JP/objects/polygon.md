## 多角形

### Description
サイズ、角の数、色、種類 を指定して図形を描画します。

### Properties

#### 横幅

* 最小 - 0
* デフォルト - 100

#### 高さ

* 最小 - 0
* デフォルト - 100

#### ライン幅

* 最小 - 0
* デフォルト - 4000

#### 角

* 最小 - 3
* デフォルト - 3

#### 色

* デフォルト - #FFFFFFFF
