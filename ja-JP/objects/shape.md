## 図形

### Description
サイズ、色、種類 を指定して図形を描画します。

### Properties

#### 横幅

* 最小 - 0
* デフォルト - 100

#### 高さ

* 最小 - 0
* デフォルト - 100

#### ライン幅

* 最小 - 0
* デフォルト - 4000

#### 色

* デフォルト - #FFFFFFFF

#### 種類

* デフォルト - 楕円
* アイテム - "楕円, 矩形"
