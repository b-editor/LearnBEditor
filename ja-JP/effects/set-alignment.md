## アライメントを設定

### Description

画像にアライメントを設定します。

### Properties

#### 水平方向のアライメント

* デフォルト - 左
* アイテム - "左, 中央, 右"

#### 垂直方向のアライメント

* デフォルト - 上
* アイテム - "上, 中央, 下"
