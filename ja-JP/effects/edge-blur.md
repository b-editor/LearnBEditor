## 境界ぼかし

### Example

![](https://raw.githubusercontent.com/b-editor/LearnBEditor/main/ja-JP/images/edge-blur.jpg)

### Description

画像の境界をぼかします。

### Properties

#### カーネルの横幅

* 最小 - 0
* デフォルト - 20

#### カーネルの高さ

* 最小 - 0
* デフォルト - 20

#### 透明度の境界をぼかす

* デフォルト - false
