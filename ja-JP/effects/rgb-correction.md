## RGB調整

### Example

![](https://raw.githubusercontent.com/b-editor/LearnBEditor/main/ja-JP/images/rgb-correction.jpg)

### Description

画像を色調を調整します。

### Properties

#### 赤

* 最大 - 255
* 最小 - -255
* デフォルト - 0

#### 緑

* 最大 - 255
* 最小 - -255
* デフォルト - 0

#### 青

* 最大 - 255
* 最小 - -255
* デフォルト - 0
