## コントラスト調整

### Example

![](https://github.com/b-editor/BEditor/raw/main/docs/example/contrast.jpg)

### Description

画像のコントラストを調整します。

### Remarks

このエフェクトはGpu処理に対応しています。

### Properties

#### コントラスト

* 最大 - 255
* 最小 - -255
* デフォルト - 0
